mediaGallery

mediaGallery(items, callback?): MediaGalleryComponentBuilder

Defined in: components/media-gallery.ts:16

Creates a media gallery component

A Media Gallery is a top-level content component that allows you to display 1-10 media attachments in an organized gallery format. Each item can have optional descriptions and can be marked as spoilers.

Parameters​

items​

APIMediaGalleryItem[]

The items in the media gallery.

callback?​

(c) => void

Optional configuration callback for the builder.

Returns​

MediaGalleryComponentBuilder

An instance of MediaGalleryComponentBuilder.
